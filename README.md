# Xeneon Edge GameGlass Widget

> [!IMPORTANT]
> As of iCUE 5.46.67, there seems to be a problem in Widget mode stealing the focus away from the game/windows you're currently on.  This means that keybinds setup in GameGlass will not reach the game correctly.  We trying to get this fixed with Corsair.

> [!IMPORTANT]
> SSO is currently not working with iCUE because their system doesn't "register" deep links back into the widget.  We're currently working on a solution that works directly within the Hub.

Official GameGlass iCUE widget, [read more about it here](https://support.gameglass.gg/en/articles/15127433-gameglass-on-the-xeneon-edge).

If any issues arise, please create an issue here or email support: [support@gameglass.gg](mailto:support@gameglass.gg)
